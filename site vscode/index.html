A aplicação

Criada com o intuito de ajudar todas as idades, Med O'clock preocupa-se com a toma diária e correta dos medicamentos dos seus utilizadores.

Com uma interface intuita e fácil de usar, a nossa aplicação permite que cada pessoa possa gerir os seus medicamentos de forma eficiente e segura.

Med O'clock disponibiliza um glossário único de medicamentos para cada utilizadores, um calendário pessoal e a disponibilização de multiplas contas num único diapositivo.

Disponibilizamos ainda a possibilidade de criar prescrições descritivas dos medicamentos, para manter-se a par dos efeitos dos seus medicamentos, facilitando as consultas com os médicos.

Perguntas frequentes

"É uma aplicação segura para os meus familiares?"

  • Sim, Med O'clock é uma aplicação segura que preza pelos utilizadores
    acima de tudo. É uma aplicação que tem como objetivo abrangir todas
    as faixas etárias para facilitar a toma de medicamentos
    de toda a população.

"O glossário é compartilhado ou pessoal?"

  • Por agora, o glossário é exclusivamente pessoal de cada utilizador,
    guardando todos os medicamentos já inseridos previamente, facilitando a
    consulta de novas prescrições e receitas.

"É possível apagar a conta?"

  • Como em qualquer aplicação que utiliza contas, Med O'clock permite a
    eliminação simples e rápida da conta se necessário e quando desejar.
    Lembre-se que ao fazer isso,os seus dados serão permanentemente
    apagados e terá que se registar novamente para voltar a utilizar
    a aplicação, sem os medicamentos inseridos antes.
